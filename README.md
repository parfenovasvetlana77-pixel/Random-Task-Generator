# Random Task Generator 🎲

**Автор:** Алексей Иванов

## Описание

Random Task Generator — это桌面ное GUI-приложение на Python для генерации случайных задач из предопределённого списка с возможностью фильтрации по типам, ведения истории и сохранения данных в JSON.

### Основные возможности

- ✅ Генерация случайной задачи с учётом выбранного фильтра
- 📋 Три типа задач: учёба, спорт, работа
- 📜 История всех сгенерированных задач с временными метками
- 🔍 Фильтрация отображаемых задач по типу
- ➕ Добавление новых задач с проверкой на пустую строку
- ✖️ Удаление существующих задач
- 💾 Автоматическое сохранение задач и истории в JSON-файлы
- 🔄 Загрузка данных при запуске
